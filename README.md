# Final_Deployment_RP2040

Deployment-focused RP2040 Arduino sketch integrating the Cryophone pipeline with ADXL, Iridium, and GNSS functionality.

## Files

- `Final_Deployment_RP2040.ino` - main deployment sketch
- `cryo_model_0.h` - embedded model data
- `cryo_pipeline_types.h` - shared pipeline structs
